## Python 檔案處理
開檔(mode:‘r’ 唯讀模式、’w’ 寫入模式（覆寫）、 ‘a’ 寫入模式（續寫），會從原本的檔案最後繼續進行寫入)  
`f = open("檔名", mode)`  
讀檔
```
f.read()
f.readlines()
```
